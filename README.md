# EcoRecicla | Site Institucional sobre Reciclagem

Site institucional responsivo desenvolvido para divulgar conteúdos sobre reciclagem, sustentabilidade e práticas ambientais conscientes.

O objetivo do projeto é educar e informar visitantes sobre a importância da reciclagem, além de facilitar o contato direto por WhatsApp e formulário de contato.

## Demonstração

Este projeto pode ser publicado gratuitamente usando o **GitHub Pages**.

Após publicar, substitua este trecho pela URL do site:

```text
https://seu-usuario.github.io/nome-do-repositorio/
```

## Objetivo do projeto

Criar uma página institucional moderna, responsiva e educativa com foco em:

- Reciclagem
- Sustentabilidade
- Consumo consciente
- Separação correta de resíduos
- Educação ambiental
- Contato direto com visitantes

## Funcionalidades

- Layout responsivo para desktop, tablet e celular
- Menu adaptado para dispositivos móveis
- Seções institucionais
- Área explicativa sobre reciclagem
- Cards com materiais recicláveis
- Botão flutuante de WhatsApp
- Integração com redes sociais
- Botões de compartilhamento para:
  - WhatsApp

  - LinkedIn
- Formulário de contato com validação
- Envio de mensagem via `mailto:`

## Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/nome-do-repositorio.git
```

2. Acesse a pasta do projeto:

```bash
cd nome-do-repositorio
```

3. Abra o arquivo `index.html` no navegador.

Também é possível usar a extensão **Live Server** no Visual Studio Code para visualizar o projeto localmente.

## Personalização

Antes de publicar o site, altere os dados abaixo no arquivo `index.html`.

### WhatsApp

Procure por:

```html
https://wa.me/5511999999999
```

Substitua pelo número real do cliente, seguindo o formato:

```html
https://wa.me/55DDDNUMERO
```

Exemplo:

```html
https://wa.me/5511988887777
```

### E-mail

Procure por:

```html
contato@ecorecicla.com.br
```

Substitua pelo e-mail real de contato.

### Redes sociais

Procure por:

```html
https://instagram.com
https://facebook.com
https://linkedin.com
```

Substitua pelos links oficiais do cliente.

## Publicando no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie os arquivos `index.html`, `style.css` e `script.js`.
3. No GitHub, acesse:

```text
Settings > Pages
```

4. Em **Branch**, selecione:

```text
main
```

5. Em seguida, clique em **Save**.

Após alguns segundos, o GitHub irá gerar o link público do site.
